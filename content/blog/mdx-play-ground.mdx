### JavaScript/TypeScript
<CodeBlock language="typescript" title="React Hook の例">
{`import { useState, useEffect } from 'react';

interface User {
  id: number;
  name: string;
  email: string;
}

const useUser = (userId: number) => {
  const [user, setUser] = useState<User | null>(null);
  const [loading, setLoading] = useState(true);
  
  useEffect(() => {
    const fetchUser = async () => {
      try {
        const response = await fetch(\`/api/users/\${userId}\`);
        const userData = await response.json();
        setUser(userData);
      } catch (error) {
        console.error('Failed to fetch user:', error);
      } finally {
        setLoading(false);
      }
    };
    
    fetchUser();
  }, [userId]);
  
  return { user, loading };
};`}
</CodeBlock>

### C++（競技プログラミング）
<CodeBlock language="cpp" title="Union-Find（素集合データ構造）">
{`#include <bits/stdc++.h>
using namespace std;

class UnionFind {
private:
    vector<int> parent, rank;
    
public:
    UnionFind(int n) : parent(n), rank(n, 0) {
        iota(parent.begin(), parent.end(), 0);
    }
    
    int find(int x) {
        if (parent[x] != x) {
            parent[x] = find(parent[x]); // 経路圧縮
        }
        return parent[x];
    }
    
    bool unite(int x, int y) {
        int px = find(x), py = find(y);
        if (px == py) return false;
        
        // Union by rank
        if (rank[px] < rank[py]) swap(px, py);
        parent[py] = px;
        if (rank[px] == rank[py]) rank[px]++;
        
        return true;
    }
    
    bool same(int x, int y) {
        return find(x) == find(y);
    }
};`}
</CodeBlock>

## 📋 リストとネスト機能

### 順序付きリスト
1. **第一段階**
   1. サブアイテム1-1
   2. サブアイテム1-2
      - さらにネストした項目
      - 別のネスト項目
   3. サブアイテム1-3
2. **第二段階**
   - 混合リスト（順序なし）
   - もう一つの項目
3. **第三段階**

### 順序なしリスト
- **プログラミング言語**
  - フロントエンド
    - JavaScript/TypeScript
    - HTML/CSS
    - React/Vue/Angular
  - バックエンド
    - Python
    - Node.js
    - Go
  - システムプログラミング
    - C/C++
    - Rust
    - Assembly
